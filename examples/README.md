# Examples

Example proxy implementations that served as references for this project.

## Files

- **GPT.py** - Example proxy for GPT service
- **Sora.py** - Example proxy for Sora service
- **nginx.conf** - Example nginx configuration
- **correr locaL.txt** - Local running instructions
- **abrir.bat** / **SORA.bat** - Windows batch files

These examples are for reference only and not part of the main DinoRank proxy.
